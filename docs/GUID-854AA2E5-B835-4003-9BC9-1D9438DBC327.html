1.1.8 How to create Lump Sensor

Overview

This section provides step by step information for Generating Lump sensor using Touch Configurator in MCC.

Note: To create a new project, follow the steps in the 1.1.2 Generate Touch Project with MCC Melody page in the case of MPLAB® Code Configurator (MCC) Melody or the 1.1.1 Generate Touch Project with MCC Harmony page in the case of MPLAB® Code Configurator (MCC) Harmony.

Creating Lump

Step 1: Open Touch Configurator

Select MCC and open Touch Configurator in Resource Management (MCC) tab

Step 2: Create Buttons, Slider and Wheel.

Add the required number of Buttons, Sliders and Wheels

Step 3: Create Lump Button

Press control key and Click sensors ( Buttons/Sliders/Wheels) to be configured as Lump sensor.(Lump can also be created by mouse movement).Now click create Create Lump Sensor .

Note: Multiple Lump sensor can be created.

Step 4: Visualize Lump Sensor

Hexagon Graphic represents Lump sensors. The lumped individual sensors will be displayed in yellow when mouse cursor is placed over the lump sensor .

Step 5: Generate Lump Sensor Code

Click Generate in Resource Management (MCC) tab.
