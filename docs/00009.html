onboard_sensor_two_button

This example demonstrates the basic touch application where the touch sensors are measured, and the touch status is indicated using LED on the Xplained board. The touch library parameters are also displayed in the data visualizer software when the hardware kit is connected through edbg/medbg virtual com port.
