qt1_mutualcap_xpro_board

This example demonstates the basic touch application where the touch sensors are measured and the touch status is indicated using LED on the QT1 extenstion board. The touch library parameters are also displayed in the data visualizer software when the hardware kit is connected through edbg/medbg vritual com port.
