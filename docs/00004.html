Running The Application
  1. Build and Program the application using the MPLAB X IDE.
  2. The Button LEDs turns ON when touch made on the respective buttons
  3. Open Atmel Data Visualizer software and set the config path to folder that contains th datastreamer scripts
    • Scripts Folder: \touch\apps\button_only_touch_support\qt1_mutualcap\firmware\src\config\sam_c21_xpro\touch\datastreamer
  4. Open serial port connection and connect to the target. Verify the connection is made successfully and the buttons touch status are displayed on the dashboard.
    • For more information, see the microchip developer page.
